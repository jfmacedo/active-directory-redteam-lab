# Active Directory Red Team Lab

This project demonstrates a simulated attack against a corporate Active Directory environment.

The goal is to understand how attackers move through Windows domains using real-world techniques.

## Lab Overview

- Kali Linux (Attacker)
- Windows Server (Domain Controller)
- Windows 10 (Client Machine)

## Lab Progress

- Virtual lab environment created
- Domain controller configured
- Active Directory installed
- Domain users created
- Windows client joined to domain
- Initial network reconnaissance performed using Nmap
